# Security

This agent is part of the UnisonOS experience layer and is intended for use on trusted desktop/VDI environments.

For overall security and privacy guidance, see:

- https://project-unisonos.github.io/reference/security/

If you discover a security issue in unison-agent-vdi, please follow the responsible disclosure guidance in the UnisonOS security docs or open a private security advisory on GitHub rather than filing a public issue.
